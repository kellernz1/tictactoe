# 🔎 Tic Tac Toe Web App

A modern web application that allows users to play **Tic Tac Toe** against another player or a smart bot, with **animated moves, confetti celebration, and a clean interface**.

---

## 🚀 Live Demo

👉 [https://kellernz1.github.io/tictactoe](https://kellernz1.github.io/tictactoe)

---

## 🚀 Features

* 🕹️ **Player vs Player** mode
* 🤖 **Player vs Bot** mode with smart Minimax AI
* 🎨 Animated X/O moves with smooth transitions
* 🎉 Confetti animation on win
* 📱 **Responsive design** (works on desktop & mobile)
* ⚡ Fast and lightweight, no external libraries needed

---

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript

## ⚙️ Installation

1. **Clone the repository**

```bash
git clone https://github.com/kellernz1/tictactoe.git
cd tictactoe
```

2. **Open the project**

* Open the `index.html` file directly in your browser
  **OR** use a local server:

```bash
npx live-server
```

---

## 📥 Usage

1. Open the application
2. Choose a game mode: **Player vs Player** or **Player vs Bot**
3. Click on a cell to place your move
4. Watch the game automatically detect **win or draw**
5. Use **Restart Game** to play again or **Return to Menu** to change mode
